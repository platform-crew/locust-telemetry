Installation
================================


Requirements
--------------------------------

.. note::
   - Python 3.9+
   - locust>=2.37.0
   - python-json-logger>=3.3.0


Using pip
--------------------------------

.. code-block:: bash

   pip install locust-telemetry


**Validate your installation**:

.. code-block:: bash

   pip show locust-telemetry

Using uv
--------------------------------

.. code-block:: bash

  uv install locust-telemetry


Validate your installation:

.. code-block:: bash

  uv show locust-telemetry


Using poetry
--------------------------------


.. code-block:: bash

  poetry add locust-telemetry


Validate your installation:

.. code-block:: bash

  poetry show locust-telemetry

--------------------------------
