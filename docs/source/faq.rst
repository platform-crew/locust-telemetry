FAQ
===

Q: Do I need to modify my Locust tests to use Locust Telemetry?
A: No, you just need to load the plugin at the start of your test script.

Q: Can I use multiple telemetry plugins together?
A: Yes, Locust Telemetry supports multiple plugins in a single run.

Q: Where are telemetry logs stored?
A: Logs are printed to stdout by default in structured JSON format.
